## Dev-сервер и просмотр

Ты работаешь на Pro (удалённо через SSH). Браузер у пользователя на Air.

### Запуск dev-сервера
```bash
python3 -m http.server 3000 &
```

### URL для пользователя
После запуска сервера, скажи пользователю открыть:
- **B2B лендинг:** `http://100.122.236.1:3000/`
- **B2C лендинг:** `http://100.122.236.1:3000/old/b2c-index.html`

> ВАЖНО: НЕ используй cmox — он здесь недоступен (SSH-сессия).
> Просто запускай dev-сервер и давай пользователю URL с Tailscale IP 100.122.236.1.
