API для тех, кто строит продукты с AI

Одна строка — и вы подключены. OpenAI-совместимый API.

Python
import openai

client = openai.OpenAI(
    base_url="https://api.neurogate.ru/v1"
)

# Smart Router сам выберет лучшую модель
response = client.chat.completions.create(
    model="auto",
    messages=[{"role": "user", "content": "Привет!"}]

Частые вопросы

Всё что нужно знать перед подключением

Как оплатить от юрлица?

Выставляем счёт, работаем через ЭДО (Контур, СБИС). НДС включён. Акты закрытия каждый месяц.

Данные в безопасности?

Промпты не используются для обучения моделей. Можем подписать NDA. Данные обрабатываются в защищённом контуре.

Сколько времени занимает подключение?

От 10 минут для команды до 1 дня для полной интеграции с API. Поможем на каждом этапе.

Есть обучение для команды?

Да, включено в тариф: видео-курс, шаблоны промптов по отделам, вебинар для команды.

Можно попробовать бесплатно?

Да, бесплатный пилот на 2 недели для команды до 10 человек. Без обязательств.

Что если нам нужна специфичная модель?

Подключим любую модель по запросу. Smart Router подберёт оптимальную по соотношению цены, скорости и качества.
