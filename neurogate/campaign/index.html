Product Snapshot

Что это

AI-агрегатор: единый кабинет с доступом к 100+ моделям — ChatGPT (GPT-4o, GPT-5), Claude (Opus, Sonnet), Midjourney, DALL-E, Sora и другим. Работает из России без VPN, интерфейс на русском.

Монетизация

Pay-as-you-go Пользователь пополняет баланс (от 300 ₽), платит только за использование. Подписок нет. Баланс не сгорает — пополнил и пользуешься когда нужно.

Оплата

Рубли: СБП, банковские карты (РФ). Для B2B: счёт от юрлица, акты.

Стратегия при малом бюджете

  • Неделя 1–2: Стратегия «Максимум кликов» с дневным лимитом. Цель — накопить данные (50+ конверсий для обучения).
  • После 50 конверсий: Переключить на «Максимум конверсий» с оплатой за конверсию (CPA).
  • Ориентиры: CPL < 300 ₽, CR лендинга > 3%, ROAS > 2x через 30 дней.

  • Устройства/гео — корректировки ставок по данным.

7-Day Optimization Plan

D1 — Запуск

  • Создать 2 кампании (A: B2C, B: B2B) по настройкам выше
  • Загрузить ключи по группам (A1–A5, B1–B2)
  • Добавить минус-слова (глобальные + по кампании)
  • Добавить объявления (минимум 2 варианта на группу для A/B)
  • Проверить UTM-метки, привязку Метрики
  • Запустить. Проверить через 2 часа: клики идут, CTR адекватный.

D2 — Первая чистка

  • Отчёт «Поисковые запросы» — добавить минус-слова (нерелевантные запросы)
  • Проверить: нет ли аномально дорогих кликов
  • Проверить: мобильный трафик — адекватный CTR/CR?

D3 — Оптимизация объявлений

  • Повторная чистка поисковых запросов
  • Если есть варианты объявлений с CTR < 2% — заменить
  • Оценить: какие группы дают клики? Перераспределить бюджет внутри кампании.
  • Добавить найденные в отчёте релевантные ключи, которых не было.

D5 — Промежуточный анализ

  • Считаем: CPL по каждой группе, средний CPC, CTR, конверсии
  • Группы с 0 конверсиями и высоким расходом — снизить ставки или пауза
  • Горячие группы (A2, A4) — если CPL хороший, можно увеличить бюджет
  • Добавить уточнения/быстрые ссылки, если ещё не добавлены

D7 — Итоги первой недели

  • Полный отчёт: расход, клики, CTR, конверсии, CPL по каждой группе
  • Решение: какие группы масштабировать, какие паузить
  • Если набрали 50+ конверсий — переключить стратегию на «Максимум конверсий»
  • Планировать: запуск РСЯ (неделя 3), ретаргетинг (неделя 3–4)
  • Обновить минус-слова (финальная чистка за неделю)
